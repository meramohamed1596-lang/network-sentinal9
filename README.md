# Intelligent Network Management System

نظام إدارة الشبكات الذكي يحتوي على:

- تسجيل دخول وتسجيل حساب للعميل
- إدخال أوامر الشبكة وفحصها
- كشف الأخطاء واقتراح تصحيح تلقائي
- حفظ لقطات (snapshots) لكل عميل
- تسجيل الأحداث (logs)
- إعدادات لتحديد فترة النسخ الاحتياطي
- تكامل مع EVE-NG لاختبار الإعدادات داخل بيئة محاكاة
- واجهة تشبه التنسيق المطلوب وتعمل بشكل تفاعلي

## المتطلبات

- Python 3.14+
- Flask
- Werkzeug

## التثبيت

1. افتح Terminal في المجلد `network sentinal1`
2. ثبِّت المتطلبات:

```powershell
pip install -r requirements.txt
```

## التشغيل

### تشغيل واجهة Flask

```powershell
python app.py
```

ثم افتح المتصفح واذهب إلى:

```
http://127.0.0.1:5000
```

### تشغيل واجهة Streamlit

```powershell
streamlit run streamlit_app.py
```

ثم افتح المتصفح واذهب إلى:

```
http://127.0.0.1:8501
```

## ملاحظات

- النظام يستخدم قاعدة بيانات SQLite في الملف `network_system.db`.
- يتم حفظ نسخة احتياطية تلقائياً لكل عميل كل 15 دقيقة بشكل افتراضي.
- لإعداد EVE-NG افتح صفحة Settings وأدخل EVE-NG URL واسم المستخدم وكلمة المرور ومسار الـ lab مثل `/Network-Sentinel.unl`.
- واجهة المستخدم مصممة لتبدو قريبة من الشكل الذي طلبته وتدعم الشركات المختلفة.
